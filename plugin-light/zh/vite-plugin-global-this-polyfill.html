GlobalThis 垫片 ​

对没有 globalThis 的环境进行 polyfill，比如 QQ 小程序。

如何使用 ​

安装

bash
pnpm add @plugin-light/vite-plugin-gen-mp-qq-app-id -D

在 vite.config.ts 中添加如下设置：

ts
import { defineConfig } from 'vite';
import { globalThisPolyfillVitePlugin } from '@plugin-light/vite-plugin-gen-mp-qq-app-id';


export default defineConfig({
  plugins: [
    globalThisPolyfillVitePlugin(),
  ],
});
