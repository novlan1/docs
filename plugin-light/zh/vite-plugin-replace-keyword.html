# 替换关键词

替换项目中的关键词。

比如想用微信小程序的分包异步加载时，可在项目中使用一个特殊字符串，然后用本插件替换成 require，来避免被 vite 打包处理。

# 如何使用

安装

pnpm add @plugin-light/vite-plugin-replace-keyword -D

在 vite.config.ts 中添加如下设置：

import { defineConfig } from 'vite';
import { replaceKeywordVitePlugin } from '@plugin-light/vite-plugin-replace-keyword';


export default defineConfig({
  plugins: [
    replaceKeywordVitePlugin({
      replaceMap: {
        requireDynamic: 'require',
      }
    }),
  ],
});

# 参数

export type IOptions = {
  // key 为 要替换的关键词，value 为 目标字符串
  replaceMap: Record<string, string>;
};
