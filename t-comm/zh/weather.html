# fetchWeatherData()

描述：

获取天气信息

参数：

返回: Promise.<Array>

天气数据

示例

fetchWeatherData().then(content => {
  console.log(content)
})

# sendWeatherRobotMsg(options)

描述：

获取天气信息并发送

参数：

参数名 类型 描述
options object

配置

options.webhookUrl string

机器人hook地址

[options.chatId] string

会话Id

[options.force] string

是否在和之前获取数据相同时，也发送

返回: Promise.<Object>

请求Promise
