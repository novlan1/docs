# readEnvVariable(key, filepath)

描述：

读取文件中环境变量的值，支持：

  • NPM_TOKEN=xxx
  • NPM_TOKEN = xxx

参数：

参数名 类型 描述
key string

环境变量的key

filepath string

保存环境变量的文件路径

返回: string

环境变量的值
