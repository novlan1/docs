ts
import { formatBite } from 't-comm';

// 不支持 tree-shaking 的项目
import { formatBite} from 't-comm/lib/bite/index';

// 只支持 ESM 的项目
import { formatBite} from 't-comm/es/bite/index';

formatBite(number) ​

描述：

格式化 bite 单位，最多保留2位小数，最大单位为BB

参数：

参数名描述
number

size bite 单位

返回:

格式化的字符

示例

typescript
formatBite(1)
// 1B

formatBite(100)
// 100B

formatBite(1000)
// 1000B

formatBite(10000)
// 9.77KB

formatBite(100000)
// 97.66KB

formatBite(1000000)
// 976.56KB

formatBite(10000000)
// 9.54MB
